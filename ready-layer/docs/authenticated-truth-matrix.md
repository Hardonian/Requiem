## Degraded State Legend

- **Unauthenticated**: middleware redirects protected pages to `/auth/signin`, API returns problem+json 401.
- **Dev verify mode**: `REQUIEM_ROUTE_VERIFY_MODE=1` and non-production runtime; middleware injects synthetic auth headers.
- **Real auth mode**: verify mode disabled; middleware requires Supabase user session.
- **Backend missing**: `REQUIEM_API_URL` absent; authenticated UI shows explicit degraded notices and local/empty states.
- **Backend unreachable**: `REQUIEM_API_URL` set but service down; route-level handlers surface degraded error responses.
- **No data**: API responds with empty collections; UI uses explicit empty-state components.
- **Engine/runtime unavailable**: engine endpoint errors/timeouts surfaced as degraded cards/problem details.
- **Forbidden**: route handlers that validate tenant auth return explicit 403/401 problem payloads.
- **Informational/static route**: no runtime dependency beyond app shell.
- **Local-only route**: controls mutate client state only; no backend mutation authority implied.
- **Source-inspected only**: route/control identified from source but not runtime exercised in this environment.
