## Visual Requirements

1. **Terminal**: Use a clean, dark-mode terminal (e.g., Catppuccin / Horizon Theme).
2. **Font**: Use a monospaced font with high legibility (e.g., JetBrains Mono, Input Mono, Roboto Mono).
3. **Resolution**: Minimum 1080p (1920x1080). 4K (3840x2160) preferred.
4. **Window Padding**: Maintain standard padding for UI elements. No cluttered background.
