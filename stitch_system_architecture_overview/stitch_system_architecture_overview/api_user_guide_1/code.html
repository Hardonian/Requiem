API Implementation Guide

Architecture Overview

Ready Layer operates as a deterministic control plane utilizing signature-based authentication and manifest-level integrity for replication protocol consistency. The architecture ensures atomicity across distributed nodes by enforcing strict sequence validation and cryptographic proofs for every state transition.

01

Authentication

All requests must be authenticated using the HMAC-SHA256 signing algorithm. Required headers include X-Ready-Layer-Signature and X-Ready-Layer-Timestamp. Signatures are valid for a 30-second window.

Signing Logic (Node.js) Copy 
const crypto = require('crypto');
const secret = process.env.READY_LAYER_SECRET;
const timestamp = Date.now().toString();
const payload = `${method}:${path}:${timestamp}`;

const signature = crypto
  .createHmac('sha256', secret)
  .update(payload)
  .digest('hex');

// Headers:
// X-Ready-Layer-Signature: ${signature}
// X-Ready-Layer-Timestamp: ${timestamp}
02

Entitlements Engine

The /v1/entitlements/verify endpoint performs deterministic validation of resource access rights against the global manifest state.

Request [POST]

{
  "resource_urn": "urn:rl:obj:8829",
  "action": "READ_REPLICATION",
  "context": {
    "node_id": "us-east-1a",
    "priority": 1
  }
}

Response [200 OK]

{
  "authorized": true,
  "token": "rl_auth_v1_f8e2...",
  "ttl": 3600,
  "integrity_hash": "6b2a...91f"
}
03

Replication Protocol

The protocol uses a strict monotonic sequence counter. Any deviation in sequence numbers results in immediate rejection to prevent state bifurcation.

Export [GET]

/v1/replicate/export?seq={last_known}

Retrieves the delta-manifest since the last known sequence ID. Includes Merkle proofs for verification.

Import [POST]

/v1/replicate/import

Submits a signed manifest block for global synchronization. Atomic commit-or-rollback behavior.

04

Error Handling

Code Status Definition & Resolution
401 Unauthorized Signature mismatch or expired timestamp. Verify system clock synchronization (NTP).
403 Forbidden Insufficient entitlements for the requested resource URN or sequence window locked.
429 Rate Limit Control plane congestion. Implement exponential backoff with jitter.
503 Out of Sync Local node sequence exceeds control plane consensus. Forced resync required.
