Get Started

Quick Start Guide

Initialize your local environment to interact with the Control Plane.

1

Install CLI

Install the `ready-cli` tool via brew or curl.

bash
brew install ready-cli
2

Initialize Policy

Create a default policy configuration file in your repo.

bash
ready policy init --template=guardrails-v1
3

Run Execution

Execute a dry-run against the local LLM proxy.

bash
ready run --dry-run --model=gpt-4-turbo
4

Verify Replay

Ensure deterministic outputs by replaying the execution ID.

bash
ready replay --id=exec_8a92b --assert=exact
5

Export Provenance

Generate a cryptographic proof of the execution chain.

bash
ready provenance export --format=json
