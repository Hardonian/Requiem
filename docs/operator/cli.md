# Reach CLI Operator Guide

The CLI interface is Reach CLI (`req`, `reach`, `requiem` binaries).

## Core commands
- `req run`
- `req replay`
- `req inspect`
- `req graph`
- `req diff`
- `req verify`
- `req policy`
- `req learn`
- `req doctor`
- `req status`

All commands accept `--json`, `--model`, `--provider`, and `--replay`.
