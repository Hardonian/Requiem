# CLI Reference (Compressed)

Operational commands:
- `requiem status`
- `requiem doctor`
- `requiem trace <id>`
- `requiem inspect <artifact>`
- `requiem explain <decision>`
- `requiem graph cas`
- `requiem graph wal`
- `requiem verify proof-pack`
- `requiem diff replay <runA> <runB>`

Replay export:
- `requiem replay export <bundle> --format graph-json`
