# Prompt Marketplace

Marketplace prompt packages live under `prompts/marketplace/`.

## Package structure

- `prompt.yaml`
- `prompt.md`
- `example_inputs.json`
- `tests/`

## Registry metadata

Marketplace entries include:
- `rating`
- `downloads`
- `compatibility`
- `runtime_requirements`

## CLI

- `rl repo prompts search [query]`
- `rl repo prompts install <prompt-id>`
- `rl repo prompts publish <prompt-id>`
