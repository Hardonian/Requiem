# Getting Started (Canonical Entry)

Primary onboarding is `docs/GETTING_STARTED.md` and `README.md`.

Fast contributor path:

1. `pnpm install`
2. `pnpm run verify:routes` (quick route/auth contract check)
3. `pnpm run dev`

Full verification path:

1. `pnpm run verify`
2. `pnpm run test`
3. `pnpm run build`
